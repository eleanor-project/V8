# Examples

The repository includes runnable examples under `examples/`. Use these to
validate a local setup or demonstrate specific capabilities.

## Quick list

- `examples/integration_example.py` - caching, observability, resilience.
- `examples/streaming_governance_demo.py` - streaming governance flow.
- `examples/gpu_example.py` - GPU capability checks.
- `examples/gpu_integration_example.py` - GPU integration with engine.
- `examples/model_configuration_examples.py` - configuration patterns.
- `examples/temporal_evolution_demo.py` - temporal policy evolution.

## Running examples

```bash
python examples/integration_example.py
python examples/streaming_governance_demo.py
```
